## ⭐ Overview
This is a web app using the [DND 5e API](https://www.dnd5eapi.com) and node.js to look up basic info <br> on Dungeons and Dragons monsters.
This includes their creature type, size, <br> hit points, and stat line (Strength, Dexterity, Consitution, etc.)

## 🔒 Requirements
* node.js
* node.js Express
* Web Browser of your choice
